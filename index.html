Q.6 What’s the difference between window vs document?

The window object represents the browser window and is the top level object in the JavaScript hierarchy. It contains the browser’s address bar, history list, menus, and other browser-specific features. The document object is a child of the window object and represents the current web page displayed in the browser window. It contains the HTML document structure and all the elements of the page.
